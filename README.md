# Assistant Intelligent RAG avec Agents

Un système complet combinant RAG (Retrieval-Augmented Generation) et des agents intelligents pour interagir avec des documents et effectuer des actions.

## Fonctionnalités

- **RAG** : Recherche et réponse basées sur vos documents (PDF, DOCX)
- **Agents intelligents** : Calculs, météo, recherche web
- **Mémoire conversationnelle** : Maintien du contexte
- **Interface Streamlit** : Interface utilisateur conviviale
